## ヒアリング結果サマリー

### 確認できた事項
- 監視対象の状態: 回答待ち、指示待ち、タスク完了
- データ取得方法: ログファイル監視
- 通知方法: macOS通知センター
- 技術スタック: Swift（macOSネイティブ）
- スコープ外: エラー通知、サウンド、バッジ、アイコン変化（将来の拡張候補）

### 追加/変更要件
- 新規プロジェクトのため、すべての要件が新規追加

### 残課題
- Claude Codeのログファイルの場所（調査必要）
- ログファイルのフォーマット（調査必要）
- 各状態を示すログエントリのパターン（調査必要）
- 通知の具体的な文言・デザイン（実装時に決定）
